# Changelog

All notable changes to WorkLane will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-18

### Added

- Initial release
- Core package with routing, permissions, and configuration
- Provider package with OpenAI, OpenRouter, and Ollama support
- Agent package with Manager, Research, Writer, Engineer, QA, Support, and Marketing agents
- Memory package for local storage
- Workflow package with summarize, draft reply, launch plan, research, create issue, create post, and video brief workflows
- Connector package with Telegram, CLI, and Webhook connectors
- CLI with init, doctor, run, agents, providers, and memory commands
- Telegram bot with /start, /agents, /help, /work, /remember, and /status commands
- API server with health, agents, workflows, and run endpoints
- Documentation for architecture, Telegram setup, providers, agents, security, and roadmap
