# Roadmap

WorkLane development roadmap.

## v0.1.0 (Current)

### Completed

- [x] Core package with routing and types
- [x] Provider package with OpenAI, OpenRouter, Ollama
- [x] Agent package with 7 agents

- [x] Connector package with Telegram, CLI, Webhook
